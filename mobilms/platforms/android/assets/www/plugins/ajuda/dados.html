Carregar:: Permite que você envie arquivos para o Moodle, para que você possa utilizá-los posteriormente. Os arquivos ficam armazenados de forma privada, pelo computador, você pode acessá-los em: Navegação >> Meu Perfil >> Meus arquivos privados.
Eventos:: apresenta os eventos dos cursos que você está matriculado.
Meus dados:: Permite visualizar o seu nome, sobrenome e e-mail. Você pode alterar a sua senha.
Meus cursos:: Exibe os curso onde você está matriculado, bem como links para o conteúdo (atividades e recursos).
Site:: abre o site do Moodle no seu navegador padrão.
Ajuda:: exibe este conteúdo.
Configurações:: Apresenta algumas opções para uso de espaço e relatar problemas e informações avançadas.
Sair:: realiza o logout e redireciona você para a tela de login.
